# Traductor de Código

Esta habilidad ayuda a mantener la base de código consistente en idioma español al traducir automáticamente comentarios y documentación técnica.

## Cuándo usarla
- Al importar librerías externas con documentación solo en inglés.
- Al revisar código heredado que necesite ser documentado para el equipo local.
- Antes de realizar un commit para asegurar que los comentarios sigan el estándar del proyecto.

## Instrucciones para el Asistente
1. Identifica los bloques de comentarios en el archivo de código.
2. Traduce el contenido al español manteniendo los términos técnicos estándar (ej. "callback", "hook", "middleware") si no tienen una traducción clara o comúnmente aceptada.
3. Asegúrate de no alterar la sintaxis del código.
